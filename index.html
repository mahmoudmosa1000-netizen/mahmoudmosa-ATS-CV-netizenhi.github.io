🎨 Digitale Folio & QR-Links
✨ Warum QR-Codes im Lebenslauf?
📱
Sofortiger Zugriff
Recruiter scannen mit dem Handy — Ihr Portfolio öffnet sich in Sekunden, ganz ohne Tippen.
🖨️
Auch auf Papier klickbar
Ausgedruckte Lebensläufe bleiben interaktiv — QR-Codes überbrücken Print und Digital.
🔗
Mehrere Links möglich
Portfolio, GitHub, Referenzprojekt, LinkedIn — jeder Link bekommt seinen eigenen QR-Code.
🎯
Professioneller Eindruck
Zeigt digitale Kompetenz und hebt Ihre Bewerbung aus der Masse heraus.
📋 So fügen Sie Links & QR-Codes ein
  1. Klicken Sie auf + Link hinzufügen unten.
  2. Geben Sie die URL ein (z.B. https://behance.net/…).
  3. Wählen Sie einen Linktext — das sind die Worte, die im Lebenslauf anklickbar erscheinen (z.B. „Mein Portfolio", „GitHub-Projekte").
  4. Aktivieren Sie Als Link anzeigen und/oder QR-Code einbetten — beides kombinierbar.
  5. Alle Links & QR-Codes erscheinen automatisch in der linken Seitenleiste Ihres Lebenslaufs.
  6. Fügen Sie so beliebig viele Links hinzu — z.B. Portfolio, GitHub, Referenz-Website, YouTube-Demo.
